| ----------------- | ----------------------------- |

### 3. Install Dependencies
Once the virtual environment is activated, install the OpenCV module:
```bash

```

### 4. Now open the src folder check out the following files and folders:

#### 2. edge_outputs: The folder where the output will contain with edge extraction

| Operating System  | Command           |
| ----------------- | ----------------- |
| **Windows**       | `py main.py`      |
| **Linux / macOS** | `python3 main.py` |
